Nested Loops


    

 Task : 1. Multiplication Table:
 - Write a function that prints a 10x10 multiplication table. The table should display the product of numbers from 1 to 10 in a grid format.
 
 
 2. Prime Numbers in a Range:
 - Write a function that finds and prints all prime numbers between 1 and 100. A prime number is only divisible by 1 and itself.
 
 
 3. FizzBuzz:
 - Write a function that prints the numbers from 1 to 100. For multiples of three, print "Fizz" instead of the number. For multiples of five, print "Buzz". For numbers which are multiples of both three and five, print "FizzBuzz".
 
 
 4. Pascal's Triangle:
 - Write a function to generate the first n rows of Pascal's triangle. Each number is the sum of the two numbers directly above it.
 
 5. Pattern Printing:
- Write a function that prints the following pattern for a given number of rows:


*


*


*




6. Matrix Transposition:
- Write a function that transposes a given 2D matrix. Transposition flips a matrix over its diagonal, switching the row and column indices of the matrix.



7. Find Common Elements in Two Arrays:
- Write a function that finds common elements in two arrays. The function should return an array of elements that appear in both input arrays.


8. Letter Frequency in a String:
- Write a function that counts the frequency of each letter in a given string. The function should ignore case and count only alphabetic characters.


9. Sum of Elements in a Diagonal of a Matrix:
- Write a function that calculates the sum of the elements in the main diagonal of a square matrix. The main diagonal runs from the top-left to the bottom-right of the matrix.


10. Array Rotation:
- Write a function to rotate a given array by k positions to the right. The function should return a new array where elements are shifted to the right by k positions.
